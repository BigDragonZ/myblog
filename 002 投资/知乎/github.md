# OpenClaw 炒股 Skill 笔记  
整理自 GitHub 开源项目，覆盖不同投资风格，可按需组合使用。  
  
---  
  
## 一、Skill 清单  
  
### 1. stock-daily-analysis-skill  
- **适合人群**：每日复盘党、决策困难户  
- **核心亮点**：  
- LLM 驱动智能分析  
- 覆盖 A/H/美股全市场  
- 技术面 + AI 操作建议三合一  
- 国内可用 DeepSeek，无需代理  
  
### 2. openclaw-stock-kb  
- **适合人群**：量化爱好者、想系统学习的小白  
- **核心亮点**：  
- 5 大量化策略库  
- 技术指标百科  
- 社媒情绪分析  
- 风控模板  
- 堪称「量化投资百科全书」  
  
### 3. stock_datasource  
- **适合人群**：专业玩家、想搭建本地金融数据库  
- **核心亮点**：  
- 13 个专业 Agent 协同  
- Tushare 完整数据接入  
- 策略回测功能  
- 企业级架构  
- **备注**：值得尝试，按需选用  
  
### 4. openclaw-stock-analyzer  
- **适合人群**：价值投资者  
- **核心亮点**：  
- 五维评分系统  
- DCF 估值模型  
- 巴菲特风格基本面分析  
  
### 5. tushare-openclaw-skill  
- **适合人群**：财报党、数据控  
- **核心亮点**：  
- Tushare 官方数据源  
- 财务报表数据超全  
  
### 6. openclaw-stock-data-skill  
- **适合人群**：短线选手  
- **核心亮点**：  
- 分钟级数据  
- 覆盖可转债 + ETF  
- 数据粒度最细  
  
### 7. stock-monitor-skill  
- **适合人群**：盯盘党  
- **核心亮点**：  
- 实时预警监控  
- 覆盖涨跌幅/成交量/均线/RSI 等指标  
- **备注**：谨慎选择，有使用门槛  
  
### 8. Openclaw-stock  
- **适合人群**：量化老手  
- **核心亮点**：  
- 全自动交易  
- 多模型支持  
- 订单流分析  
- **注意**：配置复杂，实盘需谨慎，建议先模拟  
  
### 9. stock-analysis  
- **适合人群**：美股情绪派  
- **核心亮点**：  
- 社交媒体情绪分析  
- 热门股扫描  
- **备注**：已归档项目，更新较少  
  
---  
  
## 二、组合推荐  
  
| 投资风格 | 推荐 Skill 组合 |  
|----------|-----------------|  
| 新手入门 | stock-daily-analysis-skill + openclaw-stock-kb |  
| 价值投资 | openclaw-stock-analyzer + tushare-openclaw-skill |  
| 量化交易 | stock_datasource + stock-monitor-skill |  
| 全自动党 | Openclaw-stock（先模拟盘测试） |  
| 美股玩家 | stock-analysis |  
  
---  
  
## 三、重要提醒  
1.  **安装方式**：所有 Skill 源码均在 GitHub，通过 `openclaw skills install <仓库链接>` 安装  
    2.  **前置准备**：安装前务必看清依赖和 API Key 配置  
    3.  **风险提示**：AI 建议仅供参考，投资有风险  
    4.  **实盘要求**：实盘交易前必须先回测 + 模拟盘验证  
  
    ---  
  
    ## 四、快速选型建议  
    - 刚入门：先从 **stock-daily-analysis-skill + openclaw-stock-kb** 打基础  
    - 做价值投资：优先 **openclaw-stock-analyzer + tushare-openclaw-skill**  
    - 量化/短线：从 **stock_datasource** 开始搭建数据底座，再配 **stock-monitor-skill**  
    - 全自动交易：仅推荐老手使用 **Openclaw-stock**，务必先模拟
