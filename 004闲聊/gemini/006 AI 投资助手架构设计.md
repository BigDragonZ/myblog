### 需求一：多市场低估值股票扫描 (Value Screening)

采用 **DCF (现金流折现模型)** 与 **多因子筛选** 结合。对于港、美、新市场，重点关注自由现金流（FCF）与 EV/EBITDA。

**公式参考**： DCF 模型中，企业内在价值 V 的计算公式为：

V=t=1∑n​(1+r)tCFt​​+(1+r)nTV​

其中 CFt​ 为第 t 年现金流，r 为折现率，TV 为终值。

**实现代码 (WSL2 Python 环境):**

Python

```
import yfinance as yf
import pandas as pd

def screen_undervalued(tickers):
    undervalued_list = []
    for ticker in tickers:
        stock = yf.Ticker(ticker)
        info = stock.info
        # 简单筛选逻辑：PE < 15 且 PEG < 1
        pe = info.get('forwardPE', float('inf'))
        peg = info.get('pegRatio', float('inf'))
        
        if pe < 15 and peg < 1:
            undervalued_list.append({
                'symbol': ticker,
                'pe': pe,
                'peg': peg,
                'current_price': info.get('currentPrice')
            })
    return pd.DataFrame(undervalued_list)
```

## 2. 核心 Skill 实现：结合最新消息与情感分析

为了让“交易策略师”具备处理最新消息的能力，我们需要实现一个结合搜索引擎（如 DuckDuckGo 或 Tavily）和 LLM 的新闻分析 Skill。

**情感得分模型逻辑**： 对于抓取到的 n 条新闻，赋予不同的时效权重 wi​ 和单条情感得分 si​∈[−1,1]，综合情感得分 S 为：

S=i=1∑n​wi​⋅si​

**实现代码 (WSL2 Python - `/workspace/skills/news_sentiment_skill.py`)**:

Python

```
import yfinance as yf
from duckduckgo_search import DDGS
from datetime import datetime, timezone

class NewsSentimentSkill:
    def __init__(self, llm_client):
        """
        :param llm_client: 注入本地运行或 OpenClaw 代理的 LLM 客户端
        """
        self.llm = llm_client

    def skill_analyze_sentiment_latest_news(self, ticker: str, market: str = "US") -> dict:
        """
        获取个股最新消息并生成交易情绪评分
        """
        # 1. 结合 yfinance 获取基础新闻
        stock = yf.Ticker(ticker)
        yf_news = stock.news[:3] 
        
        # 2. 结合搜索引擎获取最新突发新闻 (以规避 yfinance 数据延迟)
        query = f"{ticker} stock latest news {market}"
        ddg_news = []
        with DDGS() as ddgs:
            results = ddgs.text(query, max_results=3, timelimit='d')
            for r in results:
                ddg_news.append(r['title'] + " - " + r['body'])

        # 3. 聚合上下文
        context = f"Yahoo Finance News: {yf_news}\nWeb Search News: {ddg_news}"
        
        # 4. 调用本地 LLM 进行研判 (Agent 内部调用)
        prompt = f"""
        基于以下关于 {ticker} 的最新新闻，分析当前市场情绪。
        要求：
        1. 提取核心利好/利空事件。
        2. 给出一个量化的情感得分 (-1.0 极度看空 到 1.0 极度看多)。
        新闻数据：{context}
        """
        
        # 假设 llm.generate 是 OpenClaw 封装的方法
        analysis_result = self.llm.generate(prompt)
        
        return {
            "ticker": ticker,
            "timestamp": datetime.now(timezone.utc).isoformat(),
            "analysis": analysis_result
        }
```
